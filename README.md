## 📌 About The Project

This is a real-world business website built for an actual local bakery — 
**Bake 'N' Flake**, Bariatu, Ranchi. The goal was to help the business:

- Look more professional online
- Attract more customers
- Enable online ordering
- Grow their digital presence

This project was built as part of the **Future Interns Full Stack Web Development 
Internship — Task 3: Local Business Website & Live Pitch Project.**

## ✨ Features

### 🏠 Home Page
- Beautiful hero section with full-width bakery imagery
- Auto-scrolling offers marquee
- 3D flip card discount section with animations
- "Why Choose Us" section with scroll animations
- Customer testimonials carousel

### 🍰 Menu Page
- Live search bar with instant filtering
- Category filter tabs (All, Cakes, Pastries, Chocolates, Drinks)
- Product cards with images, descriptions and prices
- Add to Cart with animated feedback
- Persistent cart across pages using localStorage

### 🎂 Cake Customizer
- Triggered automatically when any cake is added to cart
- Live SVG cake preview that updates in real time
- Customize: Size, Flavor, Frosting Color, Toppings, Personal Message
- Live price calculator
- Confetti animation on completion

### 🎁 Gift & Occasion Planner
- 8 occasion types: Birthday, Anniversary, Baby Shower, Wedding, etc.
- Budget slider with 4 zones
- Smart gift package recommendations based on selections
- Gift wrapping and greeting card options
- Animated step-by-step flow

### 🛒 Order System
- Multi-step order modal (Details → Items → Delivery → Payment)
- Multiple payment options: UPI, Cash on Delivery, Net Banking
- Fake UPI QR code for demo
- Real-time order tracking with status updates
- Orders saved in localStorage and persist on refresh

### 📦 My Orders Page
- All placed orders shown with status badges
- Live status updates: Preparing 🟡 → On the Way 🟠 → Delivered 🟢
- Cancel order option (only when Preparing)
- Order details: ID, items, total, payment method, date & time

### 🏪 Store Locator
- 3 outlet cards: Bariatu Main, Lalpur, Kanke Road
- Open/Closed status badges
- Get Directions button linking to Google Maps
- Embedded Google Map

### 📸 Gallery Page
- Masonry grid layout
- Hover overlay with zoom effect
- Lightbox fullscreen view
- Filter by category

### 📞 Contact Page
- Contact form with Name, Email, Message
- Store address, timings, phone number
- Embedded Google Maps location
